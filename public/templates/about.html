Features :

- Data are cached so it download all data once and if you don't hard refresh everything load really fast.

- If you reload from gallery ( http://localhost:3000/category/2 for example ) it's still retrieve Data then display gallery where you're supposed to be.

- If you switch from home to a specific gallery, when you come back on home the currentPage of the pagination is saved.

- Same mechanic with gallery pagination. If you do Home -> gallery 10 -> Home -> gallery 10. if you change of category it reset the currentPage of the pagination.
